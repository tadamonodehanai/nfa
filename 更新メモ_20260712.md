# NFAプロトタイプ 更新メモ（2026.07.12）

変更ファイルは `index.html` のみ。`data.js` は変更なし（ローカル確認用に同梱）。
GitHub リポジトリの `index.html` を差し替えれば反映されます。

## 1. 関連情報のレイアウト変更（ご要望対応）

協会からの吹き出しが連続する問題を解消。**1ターン＝1つの吹き出し**に統一しました。

- 「ほかにも、こんな情報」「同じ条件で、ほかの年代を見る」などの関連案内は、
  吹き出し内の **「関連のご案内」区画（薄いグレー面＋ネイビーの小見出し）** として、
  主たる回答と明確に区別して同居
- 0件時のリカバリー応答（2連吹き出しだった）も同様に1つに統合。
  こちらの区画見出しは「次の一歩」
- 実装：`turnHtml()` を再構成、`rescueBubbles()` → `rescueParts()` に改修、CSS `.bsec` を追加

## 2. 不具合修正

- **トップの例タグ（#小学生の試合結果 など）の検索が壊れていた**
  フレーズ全体が1つの検索条件になり、無関係な結果（中学生の結果など）まで7件ヒット。
  年代入れ替えの件数も全年代「7件」と誤表示 → 自然文として解釈するよう修正（条件は「小学生」「試合結果」に分解され、正しく1件ヒット）
- **一覧カードの「出典ページを開く」がテキストのみ** → ネイビー枠のボタン化（押せる=中優先度のルール準拠）
- **出典ページの「関連する情報」が押せない箱だった** → ボタン化。
  一意に決まればそのページへ直行、決まらなければ探索面に戻り会話として続きを探す
  （例：「次節の対戦カード」→ 中学生×大会日程で探索）
- **出典ページのPDFリンクが死んでいた（#に飛ぶだけ）** → クリック時にトースト
  「サンプルのため、PDFリンクは未接続です」を表示
- **出典ページで概要文が2回重複**（冒頭とDETAILで同文） → DETAIL側を案内文に差し替え
- **トップの検索窓に送信ボタンがなかった**（Enterのみ） → ↑ボタンを追加
- モバイル幅で機能しない「＋タグ」ボタンを非表示に

## 3. L+1ロゴの差し替え（仮）

ヘッダーのL+1ロゴは、**リポジトリ初回コミット時点から埋め込みPNGデータが破損**
（紺の帯＋黒塗りのみで、ロゴの絵柄が入っていない状態。全コミット履歴で同一を確認）。
暫定として、ブランドネイビー #1D2283 のSVG再現版（L+1｜Life Plus One）に差し替え済み。
**正式なロゴ画像（PNG/SVG）を入手したら `index.html` 内の `LOGO_LP` 定数を差し替えてください。**

## 4. 動作確認済みフロー

- 例タグ → 条件分解 → 回答カード＋関連のご案内（1吹き出し）
- 年代入れ替え（小学生→中学生、件数表示も正常）
- 出典ページ：結果テーブル／関連する情報→順位表ページへ直行／PDF→トースト
- 関連「次節」→ 探索面へフォールバック（条件引き継ぎ）
- JS構文チェック（node --check）通過
